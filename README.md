# vidman

Video viewer and manager for local disk. Built with Electron for Windows 11.

## Overview

Vidman provides a simple interface to browse directories, view video thumbnails, and organize videos by dragging them between folders.

## Prerequisites

FFmpeg is required for thumbnail generation. Install it using:

```bash
winget install ffmpeg
ffmpeg -version
```

## Running

```bash
npm install
npm start
# or
npm run dev
```
